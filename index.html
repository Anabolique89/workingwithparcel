Pros / Cons

Pros Cons
Global
  • Easy
  • Does not take up a lot of space
  • Navigate to a folder and run
  • Harder to update
  • Could be the wrong version
  • You quickly end up with a lot of global packages
Local
  • npm install and everyone has the same version
  • npm install some package gives the latest version
  • Easy to share dependencies
  • Did you mess up? just go back with git, nothing is changed on the OS (or just delete the line in package.json)
  • Clearer documentation
  • Installed for each project, takes up space
  • A bit more complicated?
  • SOME tools are meant to be global
  • But that's what "npx" is for
